Bienvenido

Bienvenido a esta página, la cual tiene como objetivo proporcionarles el libro de Urantia en audio. Para quienes aún no estén familiarizados con este libro pueden leer el resumen de lo que trata. Para aquellas quienes ya sepan de que trata el libro, les explicamos a continuación cómo funciona esta página y como y donde escuchar los audios documentos cuando sean publicados.

Para escuchar los escritos tienes las siguientes opciones:

  • Spotify
  • Audible
  • Amazon music
  • Apple Podcasts

Cada una de estas plataformas tiene opción para móvil y para computadora. Te recomendamos buscar la página de la plataforma que más se acomode a tu gusto.
